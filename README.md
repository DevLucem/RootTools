This library makes it easy to access the root tools
###### All Cleaned up

### Implementation

Add Jitpack to the root build.gradle file
```
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```

Add the dependancy
```
dependencies {
	        implementation 'com.github.Lucem-Anb:RootTools:1.0.0'
	}
```

Done
